## Map/Navigation app built with React Native & Mapbox

# Tech-stack:

- React Native
- Redux & Redux Saga (store management and async calls)
- Auth -> Firebase Auth
- Backend -> NodeJS/Express
- DB -> MySQL
- Chat -> RN-Gifted-chat
